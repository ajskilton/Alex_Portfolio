Overview

This project involves the design and implementation of a Hardware and Firmware for power monintoring applications. The system includes power monitoring, safety mechanisms, and energy balancing features to optimize battery lifespan.

Features

  • at least 5% accuracy for Real-time power monitoring
  • Uart Transmition
  • Avaliable with optional Bluetooth module
  • Real-time 7 segment display

Technologies Used

  • Altium Designer for PCB design
  • AVR C for microcontroller programming
  • LT-spice for simulation
