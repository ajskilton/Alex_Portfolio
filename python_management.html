Overview

This project was orginally developed in MATLAB. Latter I ported the functionalality to pyhton as I could use libary support to write to .ics files which are compatible with all iCalender Applications.

Features

  • Fully Adjustable Schedules
  • Fully Adjustable Hours
  • Compatible with iCalender

Technologies Used

  • Altium Designer for PCB design
  • AVR C for microcontroller programming
  • LT-spice for simulation
